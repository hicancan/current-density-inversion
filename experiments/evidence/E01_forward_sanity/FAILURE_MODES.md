# Failure Modes

## 1. Sign or unit errors

Any sign, current-scale, or SI-unit mistake would invalidate downstream
generated fields and physics losses.

## 2. Canonical success is not layout realism

Wire, loop, and via sanity checks do not validate finite-width conductors,
return paths, FEM/FastHenry/COMSOL agreement, or real QDM/NV measurements.
