# 实验说明：为什么 exp06 必须存在？

## 1. 什么是反演犯罪？

在逆问题中，如果我们用同一个正演模型生成训练/测试数据，又用同一个正演模型设计反演算法或物理 loss，模型可能表现得非常好，但这种好表现只证明它适配了我们自己写的公式。

真实世界的正演过程通常含有模型误差：有限线宽、有限厚度、回流电流、材料边界、PSF、噪声、standoff 不确定等。如果模型从未见过这些偏差，真实实验时会崩。

所以必须有：

```text
解析正演训练 → 异源正演测试 → 真实实验验证
```

## 2. 本实验的防护思想

本实验用两个算子：

- low operator：模型内嵌/训练所用的理想解析算子；
- high surrogate operator：故意加入 low operator 没有建模的物理因素。

如果模型只在 low test 上很好，但在 high test 上明显变差，就说明“同源测试结果”是不可信的。

如果用少量 high samples 校准后变好，就说明多保真 domain adaptation 是必要且有效的。

## 3. 为什么不直接用 COMSOL 生成全部训练数据？

因为不现实。

真正可行的多保真路线是：

1. 用解析正演器生成大规模数据，训练基础物理能力；
2. 用少量 FEM/COMSOL/FastHenry 样本做 stress test 和微调；
3. 用真实 QDM 样品做最终闭环。

## 4. 本实验和后续路线的关系

exp06 是一道闸门：

> 在任何声称“模型能泛化到真实 QDM 数据”的论文结果之前，必须先通过异源正演器 stress test。

否则很容易把模型在 synthetic 上的高分误判为真实能力。
