# E10 PDN/KCL Distribution

Claim: `C10_pdn_kcl_distribution_need`.

This evidence package implements the first runnable generated PDN/KCL circuit
graph loop. It generates small resistive PDN graphs with VDD, GND, loads, vias,
return paths, edge resistance, solved edge currents, current closure, magnetic
forward fields, and H0/H1/H2/H3 hypothesis rows.

This is generated-domain evidence only. It is not real CAD/Gerber/GDS, external
FEM/FastHenry, or real QDM/NV validation.
