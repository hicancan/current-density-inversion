# Notes

This experiment is intentionally limited to the ideal thin-sheet, quasi-static,
free-space Fourier model. It is designed to answer the route-document question:

> Before training any network, which spatial frequencies and magnetic-field
> components are observable enough to justify inversion?

It therefore does **not** include QDM PSF, NV projection covariance, standoff
tilt, ground planes, COMSOL stress tests, or topology loss. Those belong to later
experiments.
