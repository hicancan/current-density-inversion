# E13 Multi-Height Multi-State Observability

Claims: `C02_single_plane_identifiability_boundary`, `C06_graph_hypothesis_system_identification`.

Evaluates whether multi-height sensor observations and multi-state current
excitation improve identifiability of multilayer current inversion in
generated-domain route/via/return cases.

Comparisons: single/dual/triple heights, 1/2/4 excitation states,
Bz/Bxy/Bxyz components, without/with graph prior.

Generated-domain evidence only.
