# E12 Failure Modes

- Held-out or family-hidden rows leak into feature scaling, model fitting, or
  model selection.
- Label accuracy improves while predicted currents violate KCL.
- Label accuracy improves while current closure is worse than an unconstrained
  current regressor.
- Physics-aware projection improves KCL by construction but field
  reconstruction degrades materially.
- Family-hidden performance collapses, showing only variant memorization.
- Results are overclaimed as mechanism-level explanation or real chip/QDM/CAD
  validation.
