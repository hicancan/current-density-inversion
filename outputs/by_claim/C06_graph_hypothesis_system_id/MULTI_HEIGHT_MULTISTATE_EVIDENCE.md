# E13 Multi-Height Multi-State Observability Evidence (Graph Prior)

Claim: `C06_graph_hypothesis_system_identification`.

## Summary

Generated-domain evidence shows that a graph/hypothesis prior (knowledge of which
segments belong to which layer) reduces layer misallocation error by ~1.4x across
multi-height, multi-state, and multi-component configurations.

- `experiments/evidence/E13_multi_height_multistate_observability/`

## Boundary

This is generated-domain evidence only. Graph prior reduction of layer misallocation
does not validate real CAD/Gerber/GDS graph inference or real QDM/NV via diagnosis.
